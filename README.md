# Amber Zhao Portfolio

静态作品集网站，可直接部署到 Vercel。

## 本地预览

直接打开 `index.html` 即可。

## GitHub + Vercel 部署

1. 把这个 `deploy` 文件夹里的所有内容上传到一个 GitHub 仓库根目录。
2. 在 Vercel 里选择 `Add New Project`，导入这个 GitHub 仓库。
3. Framework Preset 选择 `Other`。
4. Build Command 留空。
5. Output Directory 留空或填 `.`。
6. 点击 Deploy。

## 文件结构

- `index.html`: 网站首页
- `assets/`: 图片、视频和 logo 资源
- `vercel.json`: Vercel 静态部署配置
